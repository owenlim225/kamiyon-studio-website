PROJECT OVERVIEW

Design a modern, premium, highly polished desktop-first website for Kamiyon Studio, a Philippine-based creative technology company specializing in end-to-end game development and interactive experiences.

This website is NOT a game marketing website.

It is NOT an indie developer portfolio.

It is NOT a generic software agency website.

Instead, it should position Kamiyon Studio as a trustworthy creative technology partner capable of delivering premium-quality games for startups, Web3 companies, publishers, and businesses.

The website must immediately communicate three things:

We create beautiful games.
We can execute professionally from concept to launch.
We are the team you'd trust with your next game.

The overall impression should feel like the intersection of:

Annapurna Interactive's artistic confidence
Framer's clean layouts
Apple's spacing discipline
Nintendo's playful charm
Japanese minimalism
Modern creative agencies

The design should prioritize trust, craftsmanship, elegance, and creativity rather than looking overly "gamer" or futuristic.

FIGMA MAKE GENERATION RULES

When generating layouts:

Prioritize desktop (1440px canvas) while ensuring responsive structure.
Use Auto Layout for all sections and reusable components.
Build reusable components for navigation, buttons, cards, testimonials, team profiles, project showcases, service cards, and CTAs.
Use an 8px spacing system with consistent constraints.
Avoid placeholder layouts that look like generic SaaS templates.
Prefer asymmetrical but balanced compositions inspired by modern editorial design.
Reserve large visual areas for high-quality game screenshots.
Keep copy concise and scannable with strong visual hierarchy.
Ensure every page can scale as the studio grows, allowing more games, awards, partners, and team members without redesigning the system.

This serves as the design foundation. Every subsequent part—homepage, portfolio, services, about, and contact—should inherit these principles so the entire website feels cohesive, premium, and unmistakably Kamiyon Studio rather than a generic creative agency.
