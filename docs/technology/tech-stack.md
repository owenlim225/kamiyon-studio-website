# Purpose

This document defines the primary technologies currently used and supported by Kamiyon Studio.

Rather than serving as a rigid list of approved tools, this document provides a reference for the technologies that align with our expertise, service offerings, and long-term product vision.

As technology evolves, this document should be updated to reflect changes in industry standards and the studio's capabilities.

---

# Technology Philosophy

Kamiyon Studio is technology-agnostic.

We do not build products around specific tools.

We choose technologies that best support the project's goals, team expertise, long-term maintainability, and user experience.

The right tool is the one that enables us to deliver the best outcome.

# Technology Selection Criteria

Before adopting any new technology, Kamiyon evaluates:

* Does it solve a real problem?
* Does it improve developer productivity?
* Is it maintainable?
* Does the team have or can reasonably develop expertise?
* Does it improve the user experience?
* Is it sustainable for long-term projects?

Technology should create value rather than unnecessary complexity.
