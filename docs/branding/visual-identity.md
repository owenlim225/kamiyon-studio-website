# Purpose

This document establishes the official visual language of Kamiyon Studio.

Its purpose is to ensure consistency across every touchpoint, including:

* Logo
* Website
* Social media
* Presentations
* Merchandise
* Events
* Product branding
* Marketing materials
* User interfaces
* Packaging
* Print media

Every visual decision should reinforce Kamiyon's identity as a **playful yet professional Filipino creative technology company**.

---

# Brand Philosophy

Kamiyon's visual identity is built around one central idea:

> **Playfulness deserves craftsmanship.**

Many companies choose between looking fun or looking professional.

Kamiyon intentionally embraces both.

Our identity should feel:

* Warm
* Memorable
* Premium
* Approachable
* Modern
* Timeless

People should immediately recognize that Kamiyon creates experiences—not just software.

# Design Principles

## 1. Simplicity First

Complexity should never become visual clutter.

Layouts should feel clean, spacious, and easy to understand.

Every element must have a purpose.

---

## 2. Personality Through Detail

Small moments matter.

Rounded corners.

Gentle animations.

Thoughtful spacing.

Playful illustrations.

Friendly icons.

Premium finishes.

These details create emotional connection without overwhelming the interface.

---

## 3. Premium Without Intimidation

Kamiyon should never feel overly corporate.

Likewise, it should never feel childish.

The ideal balance is:

> **Nintendo-level approachability with premium craftsmanship.**

---

## 4. Consistency Builds Trust

Consistency creates recognition.

Every visual decision should reinforce—not reinvent—the brand.

Consistency is more valuable than novelty.

# Color Palette

## Primary Colors

### Sakura Pink

The signature brand color.

Represents:

* Creativity
* Warmth
* Curiosity
* Optimism

Used for:

* Brand highlights
* Buttons
* Calls to action
* Icons
* Accent elements

---

### Warm Ivory

Represents:

* Simplicity
* Calmness
* Accessibility

Primarily used as the main background color.

Interfaces should feel bright, welcoming, and comfortable.

---

### Charcoal

Represents:

* Professionalism
* Readability
* Stability

Used primarily for typography and interface elements.

Avoid using pure black whenever possible.

---

### Deep Indigo

Represents:

* Innovation
* Imagination
* Nighttime creativity
* Technical excellence

Used for emphasis, banners, premium sections, and darker themes.

---

## Accent Color

### Soft Gold

Used sparingly.

Represents:

* Craftsmanship
* Achievement
* Excellence

Gold should never dominate the palette.

Instead, it functions like jewelry—small details that elevate the overall presentation.

---

# Color Ratios

Recommended usage:

| Color       | Approximate Usage |
| ----------- | ----------------: |
| Warm Ivory  |               55% |
| Charcoal    |               20% |
| Sakura Pink |               15% |
| Deep Indigo |                8% |
| Soft Gold   |                2% |

This balance creates a clean visual hierarchy while preserving the brand's playful identity.

# Typography

## Primary Fonts

### Poppins

Primary UI font.

Used for:

* User interfaces
* Buttons
* Navigation
* Mobile applications
* Websites

Reasons:

* Friendly
* Highly readable
* Modern

---

### Montserrat

Primary marketing font.

Used for:

* Headlines
* Presentations
* Posters
* Marketing materials

Reasons:

* Strong hierarchy
* Excellent readability
* Contemporary appearance

---

### Beaufort for LoL

Decorative brand font.

Reserved for:

* Hero headlines
* Major campaigns
* Event branding
* Limited promotional materials

Should never replace body text.

Use sparingly to preserve its visual impact.

# Logo Philosophy

The Kamiyon logo is a stylized sakura blossom.

It represents:

* Creativity
* Growth
* New beginnings
* Craftsmanship
* Beauty through simplicity

More importantly, it symbolizes the fusion of:

* Filipino creativity
* Japanese-inspired design philosophy
* Long-term growth

The logo should always be treated as a symbol of the company's identity—not merely a graphic.

---

# The Sakura

The sakura is not used because Kamiyon is Japanese.

It represents a philosophy.

Just as cherry blossoms bloom briefly yet leave lasting memories, Kamiyon strives to create experiences that remain meaningful long after they have been played.

The sakura symbolizes:

* Growth
* Renewal
* Curiosity
* Creativity
* Appreciation for meaningful moments

* Soft gradients

* Minimalist compositions
* Kimono-inspired silhouettes
* Sampaguita-inspired decorative accents

---

# Photography Direction

Photography should feel:

* Bright
* Natural
* Human
* Authentic
* Collaborative

Avoid:

* Generic corporate stock imagery
* Overly staged office photos
* Dark, lifeless environments

Whenever possible, feature:

* Team collaboration
* Workshops
* Community engagement
* Development sessions
* Creative spaces
* Product demonstrations

People should feel like they're seeing real creators—not actors.

---

# Illustration Style

Illustrations should be:

* Rounded
* Expressive
* Friendly
* Slightly stylized
* Rich in personality

Avoid:

* Hyper-realism
* Excessive visual noise
* Aggressive aesthetics

Illustrations should evoke the same emotional warmth found in Nintendo's character design philosophy while remaining distinctly Kamiyon.

---

# Motion Design

Motion should communicate delight—not distraction.

Animations should feel:

* Soft
* Responsive
* Natural
* Playful

Examples include:

* Floating sakura petals
* Gentle fades
* Elastic button interactions
* Smooth page transitions
* Friendly loading animations

Fast, harsh animations should generally be avoided.

---

# User Interface Philosophy

Kamiyon interfaces should prioritize:

* Clarity
* Accessibility
* Readability
* Friendly interactions
* Minimal cognitive load

The interface should help users succeed rather than impress them with complexity.

Good UI often feels invisible.

---

# Merchandise Philosophy

Merchandise should be something people genuinely want to own—even if they have never worked with Kamiyon.

Potential merchandise includes:

* Apparel
* Plush toys
* Enamel pins
* Acrylic stands
* Stickers
* Desk accessories
* Art books
* Stationery

Every product should reinforce Kamiyon as a lifestyle and creative brand—not simply a software company.

---

# Visual Consistency Checklist

Before publishing any design, ask:

* Does it feel unmistakably Kamiyon?
* Is it playful without becoming childish?
* Is it professional without becoming corporate?
* Does it prioritize clarity?
* Would it still look timeless five years from now?
* Does it reflect our mission and personality?

If the answer is "no," refine the design before release.

---

Visual identity is more than aesthetics.
